Loads the given waveform library to the S1D13541 controller

Opens the given waveform binary file and sends it's content to the EPSON controller. In case this succeeds, it will update a fixed symlink location (standard_waveform_filename) with the filename given to the function. Updates to internally stored waveform file path (waveform_file_path)

Parameters
ppointer to a controller object
filenamethe path to the waveform file
Returns
success indication; PASS == 0, FAIL != 0
